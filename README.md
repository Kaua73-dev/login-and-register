Simple Login System
Overview

This is a simple login system with user registration and a home page.
If the user already has an account, they can log in normally.
If not, they can register a new account and then log in.

Features

Login Page: Allows users to access their accounts.

Register Page: Enables new users to create an account.

Home Page: Accessible after successful login.

Requirements

No installation or external dependencies are required.
Simply run the code to use the system.

How to Use

Open the project in your code editor or run it directly.

On the login page:

Enter your credentials if you already have an account.

Otherwise, go to the register page and create a new account.

After registering, return to the login page and sign in normally.

Notes

This project is intended for educational or demonstration purposes.
It focuses on simplicity and ease of understanding, not production security.
